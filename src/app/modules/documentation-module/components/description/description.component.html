Custom Button Component

<app-custom-button> is a flexible, responsive, and theme-aware button component supporting multiple variants, sizes, icons, loading/disabled states, and both button and anchor modes. It is fully compatible with dark mode and adapts to different screen sizes.

Usage

<app-custom-button variant="primary" size="medium">
  Primary Button
</app-custom-button>

<app-custom-button variant="secondary" size="large" [startIcon]="'fa fa-star'" state="loading">
  Loading Button
</app-custom-button>

<app-custom-button href="https://example.com" target="_blank" variant="tertiary">
  Link Button
</app-custom-button>

Inputs

Input Type Description
variant "primary" | "secondary" | "tertiary" Button style variant
state "default" | "disabled" | "loading" Button state
size "small" | "medium" | "large" Button size
startIcon string Optional icon class (left)
endIcon string Optional icon class (right)
href string If set, renders as <a> link
type "button" | "submit" Button type (if not link)
target string Target for link (if href)
class string Extra CSS classes

Features

  • Responsive: Adapts to all screen sizes using utility classes.
  • Dark mode: Fully theme-aware, automatically switches styles in dark mode.
  • Accessible: Supports aria-disabled and proper button/link semantics.
  • Flexible: Use as a button or link, with optional icons and loading state.
