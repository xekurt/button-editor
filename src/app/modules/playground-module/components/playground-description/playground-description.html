In this playground, you can view and interact with a customizable button component designed to demonstrate different styles and behaviors.

Playground Features

  • Live Preview: Instantly see how your changes affect the button's appearance and behavior.
  • Variant Selection: Choose from multiple button variants (such as primary, secondary, outline, etc.) to explore different visual styles.
  • Size Adjustment: Modify the button's size (small, medium, large) to fit various use cases.
  • Color Customization: Change the button's color to match your application's theme or personal preference.
  • Real-Time Feedback: All changes are reflected immediately, allowing you to experiment and learn interactively.
